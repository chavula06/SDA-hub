# SDA Hub Admin Dashboard

A simple, calm, and easy-to-use admin dashboard for SDA Hub Connect Zambia.

## Features

- **Peaceful Design**: Clean interface with calming colors inspired by Notion
- **Mobile Responsive**: Works perfectly on phones and tablets
- **Easy Uploads**: Drag-and-drop file uploads for sermons, resources, and images
- **Quick Actions**: One-click access to common tasks
- **Dark Mode**: Toggle between light and dark themes
- **Local Storage**: All data saved locally in browser

## Dashboard Sections

### 1. Dashboard Home
- Summary cards showing total sermons, prayer requests, events, and resources
- Quick action buttons for common tasks

### 2. Upload Sermon
- Upload audio files (MP3, WAV)
- Add sermon details (title, speaker, topic, description)
- Upload thumbnail images
- Drag-and-drop interface

### 3. Manage Resources
- Upload PDFs, DOCX, TXT files, and images
- Organize by categories (Bible Study, EGW Books, SDA Literature, etc.)
- Add descriptions and titles

### 4. Prayer Wall
- View and manage prayer requests
- Approve/reject requests (coming soon)
- Mark prayers as answered

### 5. Events
- Add upcoming events
- Include date, location, description
- Upload event posters

### 6. Gallery
- Upload and manage photos
- Organize event pictures
- Delete unwanted images

### 7. Donations
- Update payment information
- Edit donation messages
- Manage bank details

### 8. Settings
- Change website title and logo
- Customize colors
- Update contact information
- Add WhatsApp group links

## Getting Started

1. Open `dashboard.html` in your web browser
2. Use the sidebar to navigate between sections
3. Start uploading content and managing your site
4. All changes are automatically saved to browser storage

## Technical Details

- **HTML/CSS/JavaScript only** - No frameworks required
- **Responsive design** - Works on all screen sizes
- **Local storage** - Data persists between sessions
- **Drag-and-drop uploads** - Easy file management
- **Clean, accessible interface** - Easy for church leaders to use
